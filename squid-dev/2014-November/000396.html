On 18/11/2014 12:31 p.m., Eliezer Croitoru wrote:
> I wanted to build an ubuntu build node but there is nothing under
> the BuildFarm namespace about ubuntu. I was looking for an article
> like the "BuildFarm/CentosInstall" which will hold the basic steps
> for a squid build node.
> 
> Can it be done by someone? what are the "requirements" from a
> basic build node? - ccache - g++ , gcc , cc , c++ - vim - screen \
> tmux What others?

Not those.

These for the basic compile:
  aptitude install build-essential autoconf libtool libltdl-dev
pkg-config ccache

The rest is optional dependencies per-feature.

Admin tools like screen, vim, tmux are your choice.


If you want to build packages the .deb stuff I gave you a copy of a
while back is now been added to a git repo by Luigi and updated for
use building the current Debian packages. If you want details on
accessing that repo let me know.
