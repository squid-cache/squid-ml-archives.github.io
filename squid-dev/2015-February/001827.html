We now have both HTTP Request and Response parsers in place.

TODO list has the following steps:

* HTTP/2 frame parser
 - underway in lp:~yadi/squid/http2
 - requires HPACK decoder


* ICAP read buffer conversion to SBuf
 - underway in lp:~yadi/squid/parser-ng-icap

* ICAP response parser
 - depends on ICAP buffer conversion


* HTTP mime header parser SBuf conversion.
 - depends on ICAP buffer conversion


* ChunkedDecoder conversion to SBuf
 - depends on ICAP buffer conversion


* HTCP read buffer conversion to SBuf

* HTCP payload parser converting to HTTP Request parser
 - depends on HTCP buffer conversion


* Drop HttpMsg parser
 - depends HTCP payload parser conversion
 - depends ICAP response parser conversion


* other parsers to check/upgrade:
 - PROXY protocol parser
 - FTP native request
 - FTP response
 - Gopher response
 - Whois Response
 - URI parser
