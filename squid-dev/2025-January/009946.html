Squid v7 does not exist (yet): We have not made any v7 releases, and 
even v7 branch has not been created yet. See the following page for our 
release schedule: https://wiki.squid-cache.org/ReleaseSchedule

If you want to work with current "master" branch sources (that v7 will 
eventually be branched off), that branch is available in the official 
Squid repository located at https://github.com/squid-cache/squid

There are more details at
https://wiki.squid-cache.org/DeveloperResources/
