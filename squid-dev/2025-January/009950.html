On 2025-01-22 13:48, VEDIA Christian wrote:

> Does the latest version support either [HTTP/2] or HTTP/3?

Hi Christian,

     No, Squid v6 does not support HTTP/2+.


> are there plans to have this supported in the next release?

No, and there is not enough time to change that because the next release 
branch -- Squid v7 -- should be created (and more-or-less become 
off-limits for major refactoring changes) in about ten days:
https://wiki.squid-cache.org/ReleaseSchedule

FWIW, there are plans to have HTTP/3 and/or HTTP/2 support in Squid v8, 
but several major code changes are required _before_ we can start 
working on HTTP/2+ support, and I cannot promise anything about v8 
release today due to insufficient cooperation among core developers.


HTH,

Alex.
