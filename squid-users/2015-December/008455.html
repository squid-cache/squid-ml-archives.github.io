Use LIBOPENSSL_CFLAGS (Feel free to run ./configure --help first ).
Something like that:

./configure '--with-openssl' '--enable-ssl-crtd'
'LIBOPENSSL_CFLAGS=-I/usr/local/include/openssl -L/usr/local/lib'
