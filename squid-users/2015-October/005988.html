Antony:
It depends. and also squid has the capability to redirect traffic without Redirectors and the recommendation is still staying that if you can use some variation of squid settings to result a redirection such as deny_info use it instead of an external helper.

Sebastian:
It is very much depends on if the services are being served from the same squid instance or that one domain is on another server while the other is served by a squid(as a reverse proxy).
Once you answer the above question there is a chance you will get an answer.
