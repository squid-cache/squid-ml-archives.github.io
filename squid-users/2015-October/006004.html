I was right. You dont have a reverse-proxy. You have a explicit/forward
proxy that happens to be listening on port 80.

To make it a reverse-proxy you need to set the mode flag on http_port to
"accel".

See the documentation:
 <http://www.squid-cache.org/Doc/config/http_port/>
 <http://wiki.squid-cache.org/ConfigExamples/Reverse/VirtualHosting>
