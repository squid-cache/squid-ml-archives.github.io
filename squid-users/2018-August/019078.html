Aha. This directory cannot be created because it already exists.

Run these:

 rm -rf /usr/local/squid/var/cache/squid/ssl_db

 su squid

 /usr/lib64/squid/security_file_certgen -c \
   -s /usr/local/squid/var/cache/squid/ssl_db \
   -M 4MB
