So run /usr/libexec/ssl_crtd -c -s /var/lib/ssl_db

> *However, when I try to run this:*

> 
> *I.e. ssl_crtd is not executable from its directory and it is owned by the

./ssl_crtd -c -s /var/lib/ssl_db
instead.
