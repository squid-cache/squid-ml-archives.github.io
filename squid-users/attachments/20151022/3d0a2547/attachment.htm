    In transparent proxy setups RDP is not a problem everywhere, because
    of transparent proxy utilizes only HTTP and/or HTTPS ports.<br>
    <br>
    RDP is not uses this ports in case you are using mstsc.<br>
    <br>
    In forwarding proxy case, the best way is use NAT/firewall to bypass
    RDP connection by proxy.<br>
