<p dir="ltr">Hi Amos,</p>
<p dir="ltr">MSFT uses a handshake mechanism to sync files between enterprise and Cloud. We use squid with icap plugins to analyze data.
</p>
<p dir="ltr">The handshake is BITS_POST which is based on HTTP 1.1. When we enabled the icap plugin, the request was not going through. We were getting OTHER_METHOD response. We debugged this and fixed it where we added BITS_POST as a valid method/verb in Squid.
 We will be submitting this change for review to squid team.</p>
