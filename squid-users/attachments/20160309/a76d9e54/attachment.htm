<p class="MsoNormal">Hello all,<o:p></o:p></p>
<p class="MsoNormal"><o:p> </o:p></p>
<p class="MsoNormal">We have rebuilt the Debian (testing) package for Squid 3.5.15-1 for Ubuntu 14.04 LTS with libecap3.
<o:p></o:p></p>
<p class="MsoNormal"><o:p> </o:p></p>
<p class="MsoNormal">If you need to install the latest Squid 3.5 on Ubuntu 14.04 LTS please take a look at online repository at
<a href="http://ubuntu.diladele.com">http://ubuntu.diladele.com</a>. To use the repo run the following commands:<o:p></o:p></p>
<p class="MsoNormal"><o:p> </o:p></p>
<p class="MsoNormal"># add repo<o:p></o:p></p>
<p class="MsoNormal">echo "deb http://ubuntu.diladele.com/ubuntu/ trusty main" > /etc/apt/sources.list.d/ubuntu.diladele.com.list<o:p></o:p></p>
<p class="MsoNormal"><o:p> </o:p></p>
<p class="MsoNormal"># update the apt cache<o:p></o:p></p>
<p class="MsoNormal">apt-get update<o:p></o:p></p>
<p class="MsoNormal"><o:p> </o:p></p>
<p class="MsoNormal"># install <o:p></o:p></p>
<p class="MsoNormal">apt-get install libecap3<o:p></o:p></p>
<p class="MsoNormal">apt-get install squid-common<o:p></o:p></p>
<p class="MsoNormal">apt-get install squid <o:p></o:p></p>
<p class="MsoNormal">apt-get install squidclient<o:p></o:p></p>
<p class="MsoNormal"><o:p> </o:p></p>
<p class="MsoNormal">The following tutorial shows how we rebuilt Squid 3.5.15 on Ubuntu 14.04 LTS
<a href="http://docs.diladele.com/tutorials/build_squid_ubuntu14/index.html">http://docs.diladele.com/tutorials/build_squid_ubuntu14/index.html</a>.<o:p></o:p></p>
