<p class="MsoPlainText">Greetings everyone,<o:p></o:p></p>
<p class="MsoPlainText"><o:p> </o:p></p>
<p class="MsoPlainText">The Squid 3.5.19-1 package for Ubuntu 14.04 LTS is now available. This version is recompiled using Squid DEB source from Debian Testing with some changes required to support SSL bump / libecap3 on Ubuntu 14.04 LTS.
<o:p></o:p></p>
<p class="MsoPlainText"><o:p> </o:p></p>
<p class="MsoPlainText">* Original release notes are at <a href="http://www.squid-cache.org/Versions/v3/3.5/squid-3.5.19-RELEASENOTES.html">
http://www.squid-cache.org/Versions/v3/3.5/squid-3.5.19-RELEASENOTES.html</a> <o:p>
</o:p></p>
<p class="MsoPlainText">* The online repo is at <a href="http://ubuntu.diladele.com">
http://ubuntu.diladele.com</a> <o:p></o:p></p>
<p class="MsoPlainText">* Tutorial showing how we rebuilt Squid 3.5.19 on Ubuntu 14.04 LTS
<a href="http://docs.diladele.com/tutorials/build_squid_ubuntu14/index.html">http://docs.diladele.com/tutorials/build_squid_ubuntu14/index.html</a>
<o:p></o:p></p>
<p class="MsoPlainText">* Scripts we used to build it are at <a href="https://github.com/diladele/squid-ubuntu">
https://github.com/diladele/squid-ubuntu</a> <o:p></o:p></p>
<p class="MsoPlainText"><o:p> </o:p></p>
<p class="MsoPlainText">If you have installed previous version from this repo then please run "sudo apt-get update && sudo apt-get upgrade".  Please also check that your current squid.conf file from previous version is not overwritten.
<o:p></o:p></p>
<p class="MsoPlainText">If you are installing this version for the first time run the following commands:<o:p></o:p></p>
<p class="MsoPlainText"><o:p> </o:p></p>
<p class="MsoPlainText">    # add repo<o:p></o:p></p>
<p class="MsoPlainText">    echo "deb <a href="http://ubuntu.diladele.com/ubuntu/">
http://ubuntu.diladele.com/ubuntu/</a> trusty main" > /etc/apt/sources.list.d/ubuntu.diladele.com.list<o:p></o:p></p>
<p class="MsoPlainText"><o:p> </o:p></p>
<p class="MsoPlainText">    # update the apt cache<o:p></o:p></p>
<p class="MsoPlainText">   apt-get update<o:p></o:p></p>
<p class="MsoPlainText"><o:p> </o:p></p>
<p class="MsoPlainText">   # install <o:p></o:p></p>
<p class="MsoPlainText">   apt-get install libecap3<o:p></o:p></p>
<p class="MsoPlainText">   apt-get install squid-common<o:p></o:p></p>
<p class="MsoPlainText">   apt-get install squid <o:p></o:p></p>
<p class="MsoPlainText">   apt-get install squidclient<o:p></o:p></p>
<p class="MsoPlainText"><o:p> </o:p></p>

support@diladele.com</a> or here in the mailing list.<o:p></o:p></p>
