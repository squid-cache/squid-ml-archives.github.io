<p class="MsoNormal">Greetings all,<o:p></o:p></p>
<p class="MsoNormal"><o:p> </o:p></p>
<p class="MsoNormal">We would like to announce the next version of Web Safety ICAP web filter for Squid proxy (version 6.0.0.EF8F, built on December 12, 2017) as Release Candidate.<o:p></o:p></p>
<p class="MsoNormal"><o:p> </o:p></p>
<p class="MsoNormal">This version contains the following fixes and improvements:<o:p></o:p></p>
<p class="MsoNormal"><o:p> </o:p></p>
<p class="MsoListParagraph" style="text-indent:-.25in;mso-list:l0 level1 lfo2"><![if !supportLists]><span style="font-family:Symbol"><span style="mso-list:Ignore">·<span style="font:7.0pt "Times New Roman"">       
</span></span></span><![endif]>Added ability to block comments and related videos on YouTube.<o:p></o:p></p>
<p class="MsoListParagraph" style="text-indent:-.25in;mso-list:l0 level1 lfo2"><![if !supportLists]><span style="font-family:Symbol"><span style="mso-list:Ignore">·<span style="font:7.0pt "Times New Roman"">       
</span></span></span><![endif]>Admin UI updated from Django 1.8.17 to Django 1.11.7 (breaking change!)<o:p></o:p></p>
<p class="MsoListParagraph" style="text-indent:-.25in;mso-list:l0 level1 lfo2"><![if !supportLists]><span style="font-family:Symbol"><span style="mso-list:Ignore">·<span style="font:7.0pt "Times New Roman"">       
</span></span></span><![endif]>The backup/restore functionality was completely redesigned. It is now possible to directly import configuration backup from older version of the product into UI.<o:p></o:p></p>
<p class="MsoListParagraph" style="text-indent:-.25in;mso-list:l0 level1 lfo2"><![if !supportLists]><span style="font-family:Symbol"><span style="mso-list:Ignore">·<span style="font:7.0pt "Times New Roman"">       
</span></span></span><![endif]>Added special community build of the product. This build is based on FOSS components and is completely free. Squid proxy, Admin UI to manage it, Traffic Monitor and ClamAV eCAP antivirus are included into the preconfigured virtual
 appliance based on Ubuntu 16 <a href="http://packages.diladele.com/websafety/6.0.0.EF8F/va/ubuntu16/websafety.zip">
http://packages.diladele.com/websafety/6.0.0.EF8F/va/ubuntu16/websafety.zip</a> .<o:p></o:p></p>
<p class="MsoListParagraph" style="text-indent:-.25in;mso-list:l0 level1 lfo2"><![if !supportLists]><span style="font-family:Symbol"><span style="mso-list:Ignore">·<span style="font:7.0pt "Times New Roman"">       
</span></span></span><![endif]>Added support for haproxy’s PROXY protocol, now it is possible to know the user’s IP in cluster deployments. Policies can be applied by the IP address/range/subnet and not by only Active Directory authenticated user name/group.<o:p></o:p></p>
<p class="MsoListParagraph" style="text-indent:-.25in;mso-list:l0 level1 lfo2"><![if !supportLists]><span style="font-family:Symbol"><span style="mso-list:Ignore">·<span style="font:7.0pt "Times New Roman"">       
</span></span></span><![endif]>Kerberos REALM field is moved to UI/Squid/Auth/Kerberos. Now is possible to use NTLM or LDAP authentication without configuring any Kerberos setting at all.<o:p></o:p></p>
<p class="MsoNormal"><o:p> </o:p></p>
