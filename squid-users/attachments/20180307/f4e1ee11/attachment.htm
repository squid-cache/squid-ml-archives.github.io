<p class="MsoNormal"><o:p> </o:p></p>
<p class="MsoListParagraph" style="text-indent:-.25in;mso-list:l1 level1 lfo2"><![if !supportLists]><span style="mso-list:Ignore">-<span style="font:7.0pt "Times New Roman"">        
</span></span><![endif]>Added URL rewriter for Google Safe Browsing (Update API v4). It is now possible to check each URL for malware and malicious links. You would need to register on Google Cloud Platform and obtain your own API key. More information is at
<a href="https://developers.google.com/safe-browsing/v4/get-started">https://developers.google.com/safe-browsing/v4/get-started</a>.<o:p></o:p></p>
<p class="MsoListParagraph" style="text-indent:-.25in;mso-list:l1 level1 lfo2"><![if !supportLists]><span style="mso-list:Ignore">-<span style="font:7.0pt "Times New Roman"">        
</span></span><![endif]>Redesigned exclusions in UI. There is only one list of exclusions now. It is possible to further specify what exclusions are needed for each list entry. Supported exclusions are “Skip HTTPS decryption (SSL Bump)”, “Bypass proxy authentication”,
 “Bypass web filter and antivirus scan”, “Do not cache HTML pages” and “Bypass Google Safe Browsing”.<o:p></o:p></p>
