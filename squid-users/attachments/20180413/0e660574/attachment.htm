<p class="MsoNormal">Greetings all,<o:p></o:p></p>
<p class="MsoNormal"><o:p> </o:p></p>
<p class="MsoNormal">Next version of Web Safety web filter for Squid proxy (version 6.2.0.FD48, built on April 13, 2018, Release Candidate) is now available for download.
<o:p></o:p></p>
<p class="MsoNormal">This version contains the following fixes and improvements:<o:p></o:p></p>
<p class="MsoNormal"><o:p> </o:p></p>
<p class="MsoListParagraph" style="text-indent:-.25in;mso-list:l0 level1 lfo1"><![if !supportLists]><span style="font-family:Symbol"><span style="mso-list:Ignore">·<span style="font:7.0pt "Times New Roman"">       
</span></span></span><![endif]>Added new dynamic site categorization module. This module works on both requests and response. When categorizing requests URL, Referer and Host headers are scanned. When categorizing responses – textual contents of pages are scanned.
 Currently there are dynamic categorizer for Nudity Pornography, Adult Themes Sexuality, Drugs and Gambling categories, but more and more categorizers will be added with each release. We target to finally have all available categories covered.<o:p></o:p></p>
<p class="MsoNormal"><o:p> </o:p></p>
<p class="MsoListParagraph" style="text-indent:-.25in;mso-list:l0 level1 lfo1"><![if !supportLists]><span style="font-family:Symbol"><span style="mso-list:Ignore">·<span style="font:7.0pt "Times New Roman"">       
</span></span></span><![endif]>Redesigned and re-implemented deep content inspection engine. The speed of content inspection is a little improved. Detection is now done faster. The amount of used RAM when performing adult language detection is greatly decreased
 (approximately 10 times).<o:p></o:p></p>
<p class="MsoNormal"><o:p> </o:p></p>
<p class="MsoNormal">Pre-configured virtual appliance is available from <a href="https://www.diladele.com/download_next_version.html">
https://www.diladele.com/download_next_version.html</a> (should be run in VMWare ESXi/vSphere or Microsoft Hyper-V). GitHub repo with automation scripts we used to build this virtual appliance from stock Ubuntu 16 LTS image is at
<a href="https://github.com/diladele/websafety-virtual-appliance/tree/release-6.2.0">
https://github.com/diladele/websafety-virtual-appliance/tree/release-6.2.0</a> . <o:p>
