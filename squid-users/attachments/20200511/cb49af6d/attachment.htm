<p class="MsoNormal"><span lang="EN-US">Greetings everyone,<o:p></o:p></span></p>
<p class="MsoNormal"><span lang="EN-US"><o:p> </o:p></span></p>
<p class="MsoNormal"><span lang="EN-US">Web Safety 7.4 – ICAP web filter for Squid proxy and Admin UI for Squid Proxy is now available for production use. The following changes and improvements are included into this build.<o:p></o:p></span></p>
<ul type="disc">
<li class="MsoNormal" style="mso-margin-top-alt:auto;mso-margin-bottom-alt:auto;mso-list:l0 level1 lfo1">
<span lang="EN-US">Added ability to limit bandwidth usage per policy (bandwidth throttling is implemented using Squid’s delay pool configuration parameters). See
<a href="https://docs.diladele.com/administrator_guide_stable/web_filter/policies/bandwidth_limitation.html" target="_blank">
this documentation article</a>.<o:p></o:p></span></li><li class="MsoNormal" style="mso-margin-top-alt:auto;mso-margin-bottom-alt:auto;mso-list:l0 level1 lfo1">
<span lang="EN-US">Statistics storage optimized, the daily CSV files are packed as GZ files, reducing the storage requirements up to 10 times.<o:p></o:p></span></li><li class="MsoNormal" style="mso-margin-top-alt:auto;mso-margin-bottom-alt:auto;mso-list:l0 level1 lfo1">
<span lang="EN-US">Fixed an issue with cluster traffic log uploads from more than two cluster nodes.<o:p></o:p></span></li><li class="MsoNormal" style="mso-margin-top-alt:auto;mso-margin-bottom-alt:auto;mso-list:l0 level1 lfo1">
<span lang="EN-US">Fixed issues with rotating of Web Safety logs.<o:p></o:p></span></li><li class="MsoNormal" style="mso-margin-top-alt:auto;mso-margin-bottom-alt:auto;mso-list:l0 level1 lfo1">
<span lang="EN-US">Added support for Squid 4.11 on Ubuntu 18/Debian 10.<o:p></o:p></span></li><li class="MsoNormal" style="mso-margin-top-alt:auto;mso-margin-bottom-alt:auto;mso-list:l0 level1 lfo1">
<span lang="EN-US">Virtual appliance generation infrastructure moved to VMware ESXi 6.7 so some unknown issues might occur on older VMware vSphere/ESXi deployments.<o:p></o:p></span></li></ul>
