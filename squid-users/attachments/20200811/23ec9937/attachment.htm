STIG stands for Secure Technical Implementation Guide. It’s the standard  by which the DoD and other government entities measure whether a system, application, etc is in compliance with their protocols. SRG stands for Security Requirements Guides. They are
 both way for implementing security changes to your systems to keep them secure and compliant. <br>
