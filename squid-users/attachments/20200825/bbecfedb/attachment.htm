<p class="MsoNormal"><span lang="EN-US">The online repository with latest Squid 4.13 (rebuilt from Debian unstable with sslbump support) for Ubuntu 18 LTS 64-bit is available at squid413.diladele.com.
<o:p></o:p></span></p>
<p class="MsoNormal"><span lang="EN-US">Github repo at <a href="https://github.com/diladele/squid-ubuntu">

<o:p></o:p></span></p>
<p class="MsoNormal"><span lang="EN-US">Scripts for Ubuntu 20 and Ubuntu 16 are also available in that repo.<o:p></o:p></span></p>
<p class="MsoNormal"><span lang="EN-US"><o:p> </o:p></span></p>
<p class="MsoNormal"><span lang="EN-US">Here are simple instructions how to use the repo. For more information see readme at
<a href="https://github.com/diladele/squid-ubuntu">https://github.com/diladele/squid-ubuntu</a> .<o:p></o:p></span></p>
<p class="MsoNormal"><span lang="EN-US"><o:p> </o:p></span></p>
<p class="MsoNormal"><span lang="EN-US" style="font-size:10.0pt;font-family:"Courier New""># add diladele apt key<o:p></o:p></span></p>
<p class="MsoNormal"><span lang="EN-US" style="font-size:10.0pt;font-family:"Courier New"">wget -qO -
<a href="http://packages.diladele.com/diladele_pub.asc">http://packages.diladele.com/diladele_pub.asc</a> | sudo apt-key add -<o:p></o:p></span></p>
<p class="MsoNormal"><span lang="EN-US" style="font-size:10.0pt;font-family:"Courier New""><o:p> </o:p></span></p>
<p class="MsoNormal"><span lang="EN-US" style="font-size:10.0pt;font-family:"Courier New""># add repo<o:p></o:p></span></p>
<p class="MsoNormal"><span lang="EN-US" style="font-size:10.0pt;font-family:"Courier New"">echo "deb
<a href="http://squid413.diladele.com/ubuntu/">http://squid413.diladele.com/ubuntu/</a> bionic main" > /etc/apt/sources.list.d/squid413.diladele.com.list<o:p></o:p></span></p>
<p class="MsoNormal"><span lang="EN-US" style="font-size:10.0pt;font-family:"Courier New""><o:p> </o:p></span></p>
<p class="MsoNormal"><span lang="EN-US" style="font-size:10.0pt;font-family:"Courier New""># update the apt cache<o:p></o:p></span></p>
<p class="MsoNormal"><span lang="EN-US" style="font-size:10.0pt;font-family:"Courier New"">apt-get update<o:p></o:p></span></p>
<p class="MsoNormal"><span lang="EN-US" style="font-size:10.0pt;font-family:"Courier New""><o:p> </o:p></span></p>
<p class="MsoNormal"><span lang="EN-US" style="font-size:10.0pt;font-family:"Courier New""># install
<o:p></o:p></span></p>
<p class="MsoNormal"><span lang="EN-US" style="font-size:10.0pt;font-family:"Courier New"">apt-get install squid-common<o:p></o:p></span></p>
<p class="MsoNormal"><span lang="EN-US" style="font-size:10.0pt;font-family:"Courier New"">apt-get install squid
<o:p></o:p></span></p>
